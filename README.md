# tree-stickies-create-card-from-issue-action

This GitHub action creates a card on your TreeStickies data repository when an issue is opened.

## Inputs

### `issue_number`
### `issue_title`
### `issue_body`
### `token`
### `noteId`
### `dir`
### `x`
### `y`


## Usage

```
uses: actions/tree-stickies-create-card-from-issue-action@main
```
